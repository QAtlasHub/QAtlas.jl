# `IsingSquare` — model index

The 2D classical Ising model on the square lattice — Onsager's exactly-solved ferromagnet with a finite-temperature order-disorder transition.

```math
H = -J\sum_{\langle i,j\rangle} \sigma_i \sigma_j, \qquad \sigma_i = \pm 1
```
