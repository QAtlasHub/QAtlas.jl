!!! info "Assurance taxonomy (R1, ratified вЂ” Discussion #379)"
    Five honest levels, highest achieved tier wins:

    - рџџЈ **universality-corroborated** вЂ” agreement checked against the universality class (exponents / central charge). Strongest, regime-spanning evidence.
    - рџџў **corroborated-at-p** вЂ” an *independent* computation (finite-size ED extrapolation or a second closed form) reproduces the value at concrete parameter point(s) p.
    - рџ”µ **coherent** вЂ” an independent in-repo card exists and the value satisfies an internal invariant (sum rule / limiting case / delegation / retype, or an unrecognised/missing route), but no external value re-derives it.
    - вљЄ **cited-only** вЂ” backed only by a literature citation, or the model is ED-infeasible so a citation is the ceiling. The honest frontier вЂ” *neutral, not a penalty*.
    - рџџ  **uncorroborated-but-feasible** вЂ” `src` claims the hub, dense ED *is* feasible, yet no card checks it. **The only actionable risk.**

    Denominator split: the corroboration rate is taken over ED-*feasible* claimed hubs only. ED-infeasible models (`AKLT2D`, `ChernSimons3D`, `FibonacciAnyons`, `KagomeHeisenbergAFM`, `KitaevHoneycomb`, `PpIp2DSC`, `SYK`, `ToricCode`, `XCube`) are excluded from the risk denominator вЂ” their ceiling is the published / DMRG value.
