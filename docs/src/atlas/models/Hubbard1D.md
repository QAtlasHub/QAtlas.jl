# `Hubbard1D` — model index

The 1D Hubbard model — the minimal lattice model of interacting electrons, Lieb-Wu Bethe-ansatz solvable; a Mott insulator with spin-charge separation at half filling.

```math
H = -t\sum_{i,\sigma} \left( c^\dagger_{i\sigma} c_{i+1\sigma} + \text{h.c.} \right) + U\sum_i n_{i\uparrow} n_{i\downarrow}
```
