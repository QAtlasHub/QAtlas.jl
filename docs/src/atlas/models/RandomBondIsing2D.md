# `RandomBondIsing2D` — model index

2D square-lattice ±J random-bond Ising model (Edwards-Anderson 1975):

```
H = -Σ_{⟨i,j⟩} J_{ij} σ^z_i σ^z_j,
J_{ij} = +J  (prob. p)  or  -J  (prob. 1-p).
```
