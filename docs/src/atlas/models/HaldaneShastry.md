# `HaldaneShastry` — model index

Spin-1/2 Haldane-Shastry chain — inverse-square Heisenberg chain on the periodic chord-distance lattice (Haldane 1988, Shastry 1988):

```
H = J · Σ_{i < j}  S_i · S_j / d(i, j)^2,   d(i,j) = (N/π) sin(π(i-j)/N)
```
