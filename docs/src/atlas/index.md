!!! info "Assurance taxonomy (R1, ratified тАФ Discussion #379)"
    Five honest levels, highest achieved tier wins:

    - ЁЯЯг **universality-corroborated** тАФ agreement checked against the universality class (exponents / central charge). Strongest, regime-spanning evidence.
    - ЁЯЯв **corroborated-at-p** тАФ an *independent* computation (finite-size ED extrapolation or a second closed form) reproduces the value at concrete parameter point(s) p.
    - ЁЯФ╡ **coherent** тАФ an independent in-repo card exists and the value satisfies an internal invariant (sum rule / limiting case / delegation / retype, or an unrecognised/missing route), but no external value re-derives it.
    - тЪк **cited-only** тАФ backed only by a literature citation, or the model is ED-infeasible so a citation is the ceiling. The honest frontier тАФ *neutral, not a penalty*.
    - ЁЯЯа **uncorroborated-but-feasible** тАФ `src` claims the hub, dense ED *is* feasible, yet no card checks it. **The only actionable risk.**

    Denominator split: the corroboration rate is taken over ED-*feasible* claimed hubs only. ED-infeasible models (`AKLT2D`, `ChernSimons3D`, `FibonacciAnyons`, `KagomeHeisenbergAFM`, `KitaevHoneycomb`, `PpIp2DSC`, `SYK`, `ToricCode`, `XCube`) are excluded from the risk denominator тАФ their ceiling is the published / DMRG value.
