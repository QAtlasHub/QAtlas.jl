# `SixVertex` — model index

Classical six-vertex model on the infinite square lattice in the standard (a, b, c) parameterisation (Lieb 1967, Baxter 1982 ch. 8):

```
ω₁ = ω₂ = a,   ω₃ = ω₄ = b,   ω₅ = ω₆ = c
Δ = (a² + b² − c²) / (2 a b)
```
