# `SSH` — model index

The Su-Schrieffer-Heeger chain — a 1D dimerised tight-binding model (chiral class BDI) whose topological phase ($|w|>|v|$) carries winding number $W=1$ and protected end zero modes.

```math
H = \sum_i \left( v\, c^\dagger_{i,A} c_{i,B} + w\, c^\dagger_{i,B} c_{i+1,A} + \text{h.c.} \right)
```
