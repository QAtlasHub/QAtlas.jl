# `ExtendedHubbard1D` — model index

1D t-U-V Hubbard chain (nearest-neighbour-density extension of the standard Hubbard model):

```
H = -t Σ_{i, σ} (c†_{i,σ} c_{i+1,σ} + h.c.)
+ U Σ_i n_{i,↑} n_{i,↓}
+ V Σ_i n_i n_{i+1}.
Δ_c = (16 t² / U) ∫_1^∞ dω  √(ω² - 1) / sinh(2π t ω / U).
```
