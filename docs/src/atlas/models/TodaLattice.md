# `TodaLattice` — model index

Classical 1-D Toda lattice (Toda 1967) with standard nearest-neighbour exponential potential `V(r) = (a/b) e^{-b r} + a r - a/b`.  The defaults `a = b = 1` give the canonical textbook form `V(r) = e^{-r} + r - 1`.
